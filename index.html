Technical Projects
Coding projects that I have completed over the years.

Drum Machine Summer 2016

Using ArrayLists, JavaFX, and Java, I created a drum-machine with customizable sounds that plays at standard 4/4 time. It uses Java's Clip class to play .wav files - five default .wav sounds are provided. When multiple sounds are played at the same time, I merge the byte arrays of the played .wav files and play the newly created byte array.

BearMaps Spring 2016

Using an A* algorithm and a quadtree that holds a collection of images representing the city of Berkeley, I created a working application that returns the shortest path between any two locations in Berkeley. Also, I used a trie to implement an autofill functionality in a search bar.

Word Processor Spring 2016

Using a doubly linked list in Java to hold user-inputted text and JavaFX to render text, I recreated a word processor. It accounted for word wrapping, implemented arrow key controls, mouse clicking, and deletion.

Scheme Interpreter Fall 2015

Using Python, my partner, Andrew Martinez, and I wrote an interpreter that is able to read Scheme code. Scheme is a language that revolves around lists and pairs, so by utilizing classes that represent a pair and the “nil” object in Scheme, I created syntax reader and evaluator functions that allowed the user to type in Scheme.
